Title: Password Reset via Email

As a registered customer,
I want to reset my password via a secure email link,
so that I can regain access to my account when I forget my credentials.

Acceptance Criteria:
- User can request a password reset by entering their registered email address
- If email is registered, a reset link is sent within 60 seconds
- If email is NOT registered, show a generic message (do NOT reveal whether email exists — for security)
- Reset link expires in 15 minutes
- Reset link can only be used once
- New password must be at least 8 characters, contain 1 uppercase, 1 number, 1 special character
- User is auto-logged in after successful reset
- User receives a confirmation email after password change
